# Simple-Libray
![](simple-library.gif)

## Description
Basic library where users can add their favorite books and track their reading progress between two options. This web app also uses the localStorage Web API to persist data. 
This project is part of The Odin Project curriculum!

## Lessons Learned

- I got way more confortable with Javascript and DOM manipulation.
- Learned how to use the localStorage web API to persist user data.
